# Flip.java
Uses Math.random() and an if-else statement to print the results of a coin flip

 *  Compilation:  javac Flip.java
 *  Execution:    java Flip
 
 *  Simulate a fair coin flip and print out "Heads" or "Tails" accordingly.
 *  % java Flip
 *  Heads
 *
 *  % java Flip
 *  Heads
 *  % java Flip
 *  Tails
